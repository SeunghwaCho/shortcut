# ⌨️ Shortcut Key Collection

일상적으로 자주 사용하는 다양한 프로그램들의 주요 단축키를 정리한 곳입니다. 생산성을 높이기 위해 필수적인 단축키들을 카테고리별로 모아두었습니다.

## 🚀 지원하는 도구 (Supported Tools)

현재 다음과 같은 도구들에 대한 단축키 정보를 제공합니다. 각 링크를 클릭하면 상세 페이지로 이동합니다.

- **[IntelliJ IDEA](IntelliJ.md)**  
- **[Vim](VIM.md)**  
- **[Visual Studio Code](VSCODE.md)**  
- **[한글 (HWP)](HWP.md)**  
- **[Excel](EXCEL.md)**  

## 💡 활용 팁
- `Ctrl + F`를 사용하여 각 문서 내에서 원하는 기능을 빠르게 찾을 수 있습니다.
- 대부분의 개발 도구 단축키는 Windows/Linux 기준으로 작성되었습니다.

## 🛠️ 기여 방법
더 유용한 단축키가 있거나 잘못된 정보가 있다면 언제든지 이슈를 생성하거나 Pull Request를 보내주세요!
