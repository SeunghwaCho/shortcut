# VS Code 단축키 모음 (Windows 기준)

### 🔍 검색 및 찾기
| 단축키 | 기능 |
| --- | --- |
| `Ctrl + Shift + F` | 전체 파일에서 찾기 (Global Search) |
| `Ctrl + Shift + H` | 전체 파일에서 바꾸기 (Global Replace) |
| `Ctrl + F` | 현재 파일에서 찾기 |
| `Ctrl + H` | 현재 파일에서 바꾸기 |
| `Ctrl + P` | 파일명으로 파일 열기 |
| `Ctrl + Shift + O` | 파일 내 심볼(기호) 검색 |

### 📝 편집
| 단축키 | 기능 |
| --- | --- |
| `Shift + Alt + Down` | 아래로 줄 복사 |
| `Shift + Alt + Up` | 위로 줄 복사 |
| `Alt + Down` | 줄 아래로 이동 |
| `Alt + Up` | 줄 위로 이동 |
| `Ctrl + /` | 한 줄 주석 |
| `Shift + Alt + A` | 블록 주석 |
| `Shift + Alt + F` | 코드 포맷팅 (정렬) |
| `Ctrl + Shift + K` | 현재 줄 삭제 |
| `Ctrl + Enter` | 다음 줄에 빈 줄 삽입 |
| `Ctrl + Shift + Enter` | 이전 줄에 빈 줄 삽입 |

### 🧭 네비게이션
| 단축키 | 기능 |
| --- | --- |
| `Ctrl + B` | 사이드바 켜기/끄기 |
| `Ctrl + \` | 에디터 화면 분할 |
| `Ctrl + G` | 줄 번호로 이동 |
| `F12` | 정의로 이동 |
| `Shift + F12` | 모든 참조 확인 |
| `Alt + ← / →` | 이전/다음 커서 위치로 이동 |
| `Ctrl + Tab` | 다음 탭 파일로 이동 |

### 🚀 실행 및 디버그
| 단축키 | 기능 |
| --- | --- |
| `F5` | 디버깅 시작 / 계속 |
| `Ctrl + F5` | 디버깅 없이 시작 |
| `Shift + F5` | 디버깅 중단 |
| `F10` | Step Over (다음 줄) |
| `F11` | Step Into (메서드 안으로) |
| `Shift + F11` | Step Out (메서드 밖으로) |
| `Ctrl + Shift + Y` | 디버그 콘솔 열기 |

### 🛠️ 기타
| 단축키 | 기능 |
| --- | --- |
| `Ctrl + Shift + P` | 모든 명령 실행 (Command Palette) |
| `Ctrl + ` | 터미널 열기 |
| `Ctrl + K S` | 모든 단축키 목록 보기 |
