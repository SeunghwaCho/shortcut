

# Excel 단축키 모음 (Windows 기준)

### 📂 파일 및 기본 작업
| 단축키 | 기능 |
| --- | --- |
| `Ctrl + N` | 새 통합 문서 만들기 |
| `Ctrl + O` | 기존 문서 열기 |
| `Ctrl + S` | 저장하기 |
| `F12` | 다른 이름으로 저장하기 |
| `Ctrl + P` | 인쇄 |
| `Ctrl + Z` | 되돌리기 (Undo) |
| `Ctrl + Y` | 다시 실행 (Redo) |

### 📝 셀 편집 및 서식
| 단축키 | 기능 |
| --- | --- |
| `F2` | 활성 셀 편집 |
| `Alt + Enter` | 셀 내 줄 바꿈 |
| `Ctrl + C` / `Ctrl + V` | 복사하기 / 붙여넣기 |
| `Ctrl + Alt + V` | 선택하여 붙여넣기 |
| `Ctrl + B` / `Ctrl + U` | 굵게 / 밑줄 |
| `Ctrl + 1` | 셀 서식 창 열기 |
| `Ctrl + Shift + ~` | 일반 서식 적용 |
| `Ctrl + Shift + 1` | 숫자 서식 적용 (소수점 2자리, 콤마) |
| `Ctrl + Shift + 3` | 날짜 서식 적용 |
| `Ctrl + Shift + 4` | 통화 서식 적용 |
| `Ctrl + Shift + 5` | 백분율 서식 적용 |

### 🧭 네비게이션 및 선택
| 단축키 | 기능 |
| --- | --- |
| `Ctrl + Arrow keys` | 데이터 범위의 끝으로 이동 |
| `Ctrl + Shift + Arrow keys` | 데이터 범위의 끝까지 선택 |
| `Ctrl + Home` | 워크시트의 맨 처음(A1)으로 이동 |
| `Ctrl + End` | 데이터가 있는 마지막 셀로 이동 |
| `Shift + Space` | 행 전체 선택 |
| `Ctrl + Space` | 열 전체 선택 |
| `Ctrl + A` | 현재 영역 전체 선택 |
| `Page Up` / `Page Down` | 위/아래로 화면 이동 |

### 🧮 수식 및 데이터
| 단축키 | 기능 |
| --- | --- |
| `Alt + =` | 자동 합계 (SUM) 수식 입력 |
| `Ctrl + ;` | 현재 날짜 입력 |
| `Ctrl + Shift + :` | 현재 시간 입력 |
| `F4` | 수식 입력 중 절대/상대 참조 전환 |
| `Ctrl + Shift + L` | 필터 적용/해제 |
| `F9` | 모든 통합 문서 수식 다시 계산 |

### 📐 행 및 열 관리
| 단축키 | 기능 |
| --- | --- |
| `Ctrl + +` (Plus) | 셀/행/열 삽입 (셀 선택 후) |
| `Ctrl + -` (Minus) | 셀/행/열 삭제 |
| `Ctrl + 9` | 선택한 행 숨기기 |
| `Ctrl + 0` | 선택한 열 숨기기 |
| `Alt + Shift + Right` | 그룹 설정 |
| `Alt + Shift + Left` | 그룹 해제 |
