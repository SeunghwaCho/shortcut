# IntelliJ 단축키 모음 (Windows/Linux 기준)

### 🔍 검색 및 찾기
| 단축키 | 기능 |
| --- | --- |
| `Double Shift` | 전체 검색 (Search Everywhere) |
| `Ctrl + F` | 현재 파일에서 찾기 |
| `Ctrl + R` | 현재 파일에서 바꾸기 |
| `Ctrl + Shift + F` | 전체 파일에서 찾기 (Find in Path) |
| `Ctrl + Shift + R` | 전체 파일에서 바꾸기 (Replace in Path) |
| `Ctrl + N` | 클래스 검색 |
| `Ctrl + Shift + N` | 파일 검색 |
| `Ctrl + Alt + Shift + N` | 심볼 검색 |

### 📝 편집
| 단축키 | 기능 |
| --- | --- |
| `Ctrl + D` | 줄 복제 (Duplicate Line) |
| `Ctrl + Y` | 줄 삭제 (Delete Line) |
| `Ctrl + Alt + L` | 코드 자동 정렬 (Reformat Code) |
| `Ctrl + Alt + O` | 최적화된 Import (Optimize Imports) |
| `Alt + Enter` | 빠른 수정 (Show Context Actions) |
| `Ctrl + /` | 줄 단위 주석 (Line Comment) |
| `Ctrl + Shift + /` | 블록 단위 주석 (Block Comment) |
| `Ctrl + Shift + U` | 대문자/소문자 전환 |
| `Alt + J` | 다음 동일한 단어 선택 (Add Selection for Next Occurrence) |

### 🧭 네비게이션
| 단축키 | 기능 |
| --- | --- |
| `Ctrl + E` | 최근 열었던 파일 목록 |
| `Ctrl + B` 또는 `Ctrl + Click` | 정의로 이동 (Go to Declaration) |
| `Ctrl + Alt + B` | 구현체로 이동 (Go to Implementation) |
| `Alt + F12` | 터미널 창 열기 |
| `F12` | 이전에 사용한 도구 창으로 이동 |
| `Esc` | 에디터로 포커스 이동 |

### 🚀 실행 및 디버그
| 단축키 | 기능 |
| --- | --- |
| `Shift + F10` | 실행 (Run) |
| `Shift + F9` | 디버그 (Debug) |
| `Ctrl + Shift + F10` | 현재 컨텍스트 실행 |
| `F8` | Step Over (다음 줄) |
| `F7` | Step Into (메서드 내부로) |
| `Shift + F8` | Step Out (메서드 밖으로) |
| `Alt + F8` | 표현식 계산 (Evaluate Expression) |

### 🛠️ 리팩토링
| 단축키 | 기능 |
| --- | --- |
| `Shift + F6` | 이름 변경 (Rename) |
| `Ctrl + Alt + M` | 메서드 추출 (Extract Method) |
| `Ctrl + Alt + V` | 변수 추출 (Extract Variable) |
| `Ctrl + Alt + C` | 상수 추출 (Extract Constant) |
| `Ctrl + Alt + P` | 파라미터 추출 (Extract Parameter) |
