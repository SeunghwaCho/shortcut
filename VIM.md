# VIM 단축키 모음 (Normal Mode 기준)

### 🔍 검색 및 찾기
| 단축키 | 기능 |
| --- | --- |
| `/keyword` | 앞에서부터 키워드 검색 |
| `?keyword` | 뒤에서부터 키워드 검색 |
| `n` | 다음 검색 결과로 이동 |
| `N` | 이전 검색 결과로 이동 |
| `:%s/old/new/g` | 파일 전체에서 단어 교체 |
| `:%s/old/new/gc` | 확인하며 파일 전체에서 단어 교체 |

### 📝 편집
| 단축키 | 기능 |
| --- | --- |
| `i` | 현재 커서 위치에서 입력 모드로 전환 |
| `a` | 현재 커서 다음 위치에서 입력 모드로 전환 |
| `o` | 다음 줄에 새로운 줄 삽입 후 입력 모드로 전환 |
| `x` | 커서 위치의 문자 삭제 |
| `dd` | 현재 줄 삭제 (잘라내기) |
| `yy` | 현재 줄 복사 |
| `p` | 붙여넣기 |
| `u` | 실행 취소 (Undo) |
| `Ctrl + r` | 다시 실행 (Redo) |

### 🧭 네비게이션
| 단축키 | 기능 |
| --- | --- |
| `h, j, k, l` | 왼쪽, 아래, 위, 오른쪽 이동 |
| `w` | 다음 단어의 시작으로 이동 |
| `b` | 이전 단어의 시작으로 이동 |
| `G` | 파일의 마지막 줄로 이동 |
| `gg` | 파일의 첫 번째 줄로 이동 |
| `0` | 줄의 맨 앞으로 이동 |
| `$` | 줄의 맨 뒤로 이동 |

### 🚀 저장 및 종료
| 단축키 | 기능 |
| --- | --- |
| `:w` | 저장 |
| `:q` | 종료 |
| `:wq` | 저장 후 종료 |
| `:q!` | 저장하지 않고 강제 종료 |
| `:x` | 저장 후 종료 (수정 사항이 있을 때만) |
