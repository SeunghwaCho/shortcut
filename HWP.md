# 한글(HWP) 단축키 모음

### 🔍 찾기 및 바꾸기
| 단축키 | 기능 |
| --- | --- |
| `Ctrl + F` | 찾기 |
| `Ctrl + H` | 찾아 바꾸기 |
| `F2` | 다시 찾기 |

### 📝 글자 및 문단 편집
| 단축키 | 기능 |
| --- | --- |
| `Alt + L` | 글자 모양 설정 |
| `Alt + T` | 문단 모양 설정 |
| `Ctrl + B` | 글자 진하게 |
| `Ctrl + U` | 밑줄 |
| `Ctrl + I` | 기울임 |
| `Alt + Shift + E` | 글자 크게 |
| `Alt + Shift + R` | 글자 작게 |
| `Alt + Shift + K` | 자간 넓게 |
| `Alt + Shift + J` | 자간 좁게 |
| `Alt + Shift + Z` | 줄 간격 넓게 |
| `Alt + Shift + A` | 줄 간격 좁게 |
| `Ctrl + C` | 복사하기 |
| `Ctrl + V` | 붙여넣기 |
| `Ctrl + Z` | 되돌리기 |
| `Ctrl + Shift + Z` | 다시 실행 |

### 🧭 문서 관리 및 서식
| 단축키 | 기능 |
| --- | --- |
| `Alt + N` | 새 문서 |
| `Alt + O` | 불러오기 |
| `Alt + S` | 저장하기 |
| `Alt + V` | 다른 이름으로 저장하기 |
| `Alt + P` | 인쇄 |
| `F7` | 편집 용지 설정 |
| `Alt + G` | 찾아가기 (페이지, 줄 등) |

### 📊 표 편집
| 단축키 | 기능 |
| --- | --- |
| `Ctrl + N, T` | 표 만들기 |
| `F5` | 셀 선택 (1회: 현재 셀, 2회: 연속 선택, 3회: 전체 선택) |
| `L` | 셀 테두리 배경 (셀 선택 상태에서) |
| `C` | 셀 배경 (셀 선택 상태에서) |
| `M` | 셀 합치기 |
| `S` | 셀 나누기 |
| `Ctrl + 화살표` | 셀 크기 조절 |
| `Alt + 화살표` | 인접한 셀과 함께 크기 조절 |
